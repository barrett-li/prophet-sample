项目详细资料

1. 项目名称: 风险管理平台开发脚手架

2. 项目代号: Prophet

3. leader: 韦海晗

4. 成员: 伍军军、李文杰、张奇、沈坚、谢立峰

5. 项目介绍:
	Prophet以Spring Framework为核心，整合Java社区的众多开源项目。
	Prophet的目标是为了构建风险管理平台提供技术基础和系统开发脚手架，项目的初期定位是规范风险管理平台的技术方向，为风险管理平台的开发建立一个起点。
	项目代号，先知(prophet)。先知是受神启示而说话的人或作为表达神的意志的阐释者；是对人类社会或自然科学的大事有较早了解或准确预言（这些预言必须是准确的）的人。
	Prophet，隐喻我们致力于为客户提供优质金融风险预警服务。
