jar包说明
--------------------------------

1. build目录里是项目编译时、单元测试时必须的jar包。

2. runtime目录里的jar包，既是编译时必须的，也是项目运行时必须的。
web应用最终用的是此目录里的jar包。jar说明如下:

- Acegi Security 1.0.7 (http://www.acegisecurity.org)
* acegi-security-1.0.7.jar

- AspectJ Weaver Classes (http://www.eclipse.org/aspectj)
* aspectjrt.jar
* aspectjweaver.jar

- Code Generation Library (http://cglib.sourceforge.net)
* cglib-nodep-2.1_3.jar

- Apache的common系列包 (http://jakarta.apache.org/commons)
* commons-*.jar

- IBM DB2 Universal JDBC driver (http://www.ibm.com/db2)
* db2jcc_*.jar

- XML工具包 (http://www.dom4j.org)
* dom4j-*.jar

- DWR 2.0 (http://directwebremoting.org/dwr)
* dwr.jar

- Ehcache 1.2.4 (http://ehcache.sourceforge.net)
* ehcache-1.2.4.jar

- FreeMarker是一个Java模版语言，它是 JSP 的绝佳替代方案 (http://freemarker.org)
* freemarker.jar

* ibatis-*.jar

* javamail.jar

* javassist.jar

* jcommon-1.0.14.jar
